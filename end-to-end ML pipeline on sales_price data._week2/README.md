# 🚗 End-to-End ML Pipeline: Tesla Deliveries

This project demonstrates a fully modularized, production-ready machine learning pipeline for predicting and forecasting Tesla vehicle deliveries, complete with an interactive Streamlit dashboard.

## 📁 Project Architecture
```text
end-to-end ML pipeline on sales_price data._week2/
├── app.py                     # Streamlit frontend dashboard (Interactive UI)
├── model/
│   ├── train_model.py         # Full ML training pipeline (XGBoost, RF, GridSearchCV)
│   └── forecast.py            # Time-Series Forecasting script (Holt-Winters)
├── data/
│   └── tesla_data.csv         # Raw dataset (Downloaded via kagglehub)
├── utils/
│   ├── preprocessing.py       # Missing value imputation & IQR Outlier capping
│   └── feature_engineering.py # Lags, rolling means, YoY growth features
├── outputs/                   # Generated backend artifacts
│   ├── forecast_plot.png      
│   ├── model_metrics.csv      
│   ├── feature_importances.csv
│   └── best_params.json
├── requirements.txt           # Python Dependencies
└── README.md                  # Project Documentation
```

## 🚀 Setup & Execution

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run the Backend Training Pipeline
This script downloads the data, preprocesses it, trains multiple algorithms, performs cross-validated hyperparameter tuning on the best model, and exports the final model `.pkl` alongside key artifacts.
```bash
python model/train_model.py
```

### 3. Generate Forecasts
This script applies Holt-Winters Exponential Smoothing to handle the strong quarterly seasonality and extrapolates a 12-month delivery forecast.
```bash
python model/forecast.py
```

### 4. Launch the Interactive Dashboard
The frontend reads the artifacts generated by the backend to provide a fast, dynamic, text-free UI displaying deep EDA, Outlier Analysis, ML metrics, and Live Forecasting.
```bash
streamlit run app.py
```

## 🧠 Key Features
- **Dynamic Navigation:** Clean sidebar navigation.
- **Advanced EDA:** Correlation Heatmaps and Scatter Matrices.
- **Data Quality:** Real-time visual comparison of data before and after IQR outlier capping.
- **ML Transparency:** Hyperparameter configurations and Feature Importance tracking.
- **Extrapolation:** Safe, realistic future predictions utilizing Exponential Smoothing.
